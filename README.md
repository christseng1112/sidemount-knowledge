# Quartz v5

## 本站維護（sidemount-knowledge）

- **發布**：`.\publish.ps1` — 以 robocopy 同步 vault → content/，commit 並 push，GitHub Actions 自動部署。
- **連結健檢**：`.\check-links.ps1` — 掃描 vault 全部引用連結、回報失效者（已知反爬 403 網域列白名單）；建議每季或潛旅前手動跑一次，異常輸出至 `link-check-failures.csv`。
