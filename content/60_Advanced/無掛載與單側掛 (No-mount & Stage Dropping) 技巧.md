# 無掛載與單側掛 (No-mount & Stage Dropping Techniques)

在洞穴探險與沉船內部探索中，潛水員常會遇到寬度或高度小於身體加雙瓶投影面積的「極端限制空間（Major Restrictions）」。此時，傳統掛載方式已無法通過，潛水員必須採用最極限的側掛操作：**無掛載（No-mount）**與**暫時性卸除氣瓶（Stage Dropping）**技術 [1][2]。

本篇將詳述 No-mount 限制空間穿越 SOP、安全風險管理、Stage Dropping 放置協定，以及單側卸瓶後的平衡修正。

---

## 🧗 1. 無掛載 (No-mount) 限制空間穿越技巧

No-mount 指**氣瓶與潛水員身體的背帶、彈性繩完全解除連結，僅靠潛水員雙手推動前行**的極限穿越技術 [1][2]。

```
【No-mount 穿越動作流程】
 1. 窄縫前懸停 ───> 2. 瓶底/瓶首解鎖 ───> 3. 氣瓶前後縱隊前推 ───> 4. 梭子式挪動穿越
   在限制空間前        解開雙側瓶底快拆扣；  雙手抓住瓶頸，將兩瓶      先將氣瓶推過窄縫；
   建立中性浮力。      釋放瓶首 Bungee。     呈前後縱隊推在正前方 [2]。 身體緊跟爬行通過 [3]。
```

### 🟢 動作要領 (No-mount SOP)
1.  **窄縫前評估**：在限制空間入口前建立中性浮力懸停，確認前方通道尺寸（必須大於人體自身厚度與氣瓶直徑） [2]。
2.  **解除掛載**：
    *   先解開左、右側氣瓶底部的 Bolt snaps。
    *   接著將左、右側的瓶首 Bungee（彈性繩）從閥手輪上拉開。此時，氣瓶與背帶完全脫離 [1][2]。
3.  **前後縱隊前推 (Forward Pushing)**：
    *   雙手死死抓住兩支氣瓶的瓶頸與一級頭處。
    *   **將兩支氣瓶在身體中線上排成前後縱隊（一前一後）**，將橫向投影寬度降到最低 [2][3]。
4.  **梭子式穿越 (Squeezing)**：
    *   雙手將縱隊氣瓶推入窄縫，手臂伸直。
    *   雙腿使用極細微的腳踝踢動，或是用另一隻手手摸岩壁/拉導線方式，像梭子一樣將身體挪動擠過窄縫 [3][4]。

---

### ⚠️ No-mount 安全風險管理
No-mount 屬於特種極限技術，伴隨極高的生命風險，必須嚴格遵守以下安全防線 [2][4]：
*   **一級頭撞擊防護**：前推時，一級頭完全暴露在外。潛水員的雙手手掌必須**護住一級頭與手輪**，防止撞擊通道頂部岩石導致O環爆裂或氣閥被意外關閉。
*   **氣瓶失手丟失預防**：在無連結狀態下，一旦失手氣瓶滑落（如在深井或強流中），潛水員將瞬間失去所有氣源。因此，**手腕必須配備備用尼龍繩圈（Wrist Lanyards）**，在解鎖後臨時套住氣瓶閥，確保即使失手氣瓶也不會脫離可及範圍 [2][4]。

## 📦 2. 暫時性卸除氣瓶 (Stage Dropping) 技巧

在長距離探索中，為了減輕返程阻力、或是為了通過前方長段的 No-mount 窄道，潛水員會將攜帶的 Stage（旅行瓶）暫時解鎖，固定在主導線上，只帶主氣瓶前行，回程時再取回 [4][5]。

```
                 [ 主引導線 (Guideline) ]
                            │
                       [雙頭快拆扣]
                            │
               [個人識別線標 (Cookie / Marker)]
                            │
               [Dropped 鋁瓶 (閥門朝上)]  <--- 固定於導線上，防被沖走
```

### 📋 Stage Dropping 安全協定 (Safety Protocol)
1.  **物理定位防沖走 (Line Attachment)**：
    *   卸下的氣瓶**絕對不能直接放在地上**。
    *   必須使用雙頭快拆扣將氣瓶的 Top clip **牢牢鎖定在主引導線上**。這能防止被氣泡吹動、水流沖走或因揚沙（Silt-out）能見度歸零時找不到氣瓶 [4][5][6]。
2.  **個人線標識別 (Personal Markers)**：
    *   在氣瓶固定處的導線上，必須掛上潛水員個人的**非定向線標（Cookie / Marker）**。這能向後續的其他潛水小組證明「這支氣瓶是有主的，且屬於當前在洞內探索的小組」，防止被他人誤當作廢棄瓶收走 [5][6]。
3.  **氣體計劃安全底線 (Gas Calculation)**：
    *   卸下 Stage 前，潛水員必須親自確認：**當前身上剩餘的主氣瓶氣量，足以支持接下來的探索、回程，並包含 Rock Bottom 應急保留**。
    *   絕對不能把返回水面的Rock bottom安全氣量留在 dropped 氣瓶中 [4][6]。
4.  **防泥沙淤積 (Silt Prevention)**：
    *   放置氣瓶時，**一級頭與二級頭必須朝上（遠離地面）**，防止其掉入底部的淤泥中。
    *   如果泥沙跑進二級頭咬嘴，回程取用時一旦吸氣會引發嚴重的 Freeflow 故障 [6]。

---

## ⚖️ 3. 單側卸瓶 (Single-sided Mount) 後的平衡修正

在 Stage Dropping 後，潛水員的身體會從對稱重量（如左側掛 Stage，右側空置）瞬間轉變為非對稱狀態。
*   此時，左側的拉扯力消失。潛水員需要釋放右側的補償配重（若有配置），或重新收緊左側核心肌群以維持 Trim。
*   詳細的水中非對稱平衡微調，請參考 **[[單氣瓶側掛配置與平衡技巧|單氣瓶側掛配置與平衡]]** 文檔 [6][7]。
