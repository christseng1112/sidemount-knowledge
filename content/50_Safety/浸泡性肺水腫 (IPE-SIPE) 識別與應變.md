# 浸泡性肺水腫 (IPE/SIPE) 識別與應變 (Immersion Pulmonary Edema)

**浸泡性肺水腫（Immersion Pulmonary Edema, IPE；游泳型稱 SIPE）** 是近年潛水醫學高度關注的猝發風險：水下**呼吸困難、咳血痰、如「肺部進水」的窒息感**，卻常被誤判為嗆水、面鏡進水或恐慌，延誤處置可致命。本庫已涵蓋 DCS 與失溫，但 IPE 機轉與前兩者完全不同、且**與深度無關、健康者亦可能發作**，本篇補齊這塊缺口。

> ⚠️ **安全聲明**：本篇為識別與現場應變之衛教複習，**不能取代潛水醫學診療**。任何水下/出水後不明呼吸困難，一律當醫療緊急事件處理並儘速就醫。

---

## 🫁 1. 機轉：為什麼「泡在水裡」就會肺積水

IPE **不是嗆水、不是溺水，而是肺泡微血管滲液** [1][2]：

* **浸泡 + 冷水**使體表血管收縮，周邊血液被擠回胸腔核心 → **胸腔血量與肺動脈壓上升**。
* 若疊加**運動費力、高血壓、心肌順應性下降**，肺微血管壓力超過臨界 → 血漿滲入肺泡 → 換氣面積被液體佔據 [1][3]。
* **與深度無關**：任何浸泡深度都可能發作，不像 DCS/氮醉隨深度加劇 [1]。

---

## 🚨 2. 徵兆識別（最易被誤判的一項）

| 徵兆 | 說明 |
| :--- | :--- |
| **呼吸困難、喘不上氣** | 核心症狀；靜止仍喘、越潛越喘 [1][2] |
| **咳嗽、咳粉紅色/血絲泡沫痰** | 肺泡滲液的特徵性表現（hemoptysis）[1][2] |
| **胸悶、「肺部灌水」窒息感** | 潛水員常自述「像在水中呼吸」[1] |
| **不成比例的極度疲倦、無力** | 換氣效率驟降所致 [1] |
| **調節器供氣正常卻仍缺氧感** | **鑑別關鍵**：氣源沒問題 → 排除 OOG/Freeflow 後仍喘，高度懷疑 IPE |

> 📌 **鑑別要點**：先依 [[側掛調節器故障應變與漏氣排除]] 與 [[氣源耗盡 (OOG) 與長管分享程序]] 排除「氣體/裝備」因素；若供氣確認正常卻持續喘咳，**不要當恐慌或嗆水安撫**，當 IPE 處置。

## 🧯 3. 現場應變 SOP [1][4]

1.  **立即示意潛伴、中止潛水**：IPE 會快速惡化，越早出水越好。
2.  **受控上升，不可衝出水面**：仍須依 [[上升速率與安全停留標準]] 控制速率；若有減壓義務須權衡（減壓 vs 缺氧惡化，以生命優先，並儘速就醫）。
3.  **減少一切費力**：出水後停止游動/搬裝備——**運動費力會加重滲液**。必要時由隊友協助拖帶（見 [[意識喪失潛水員救援與拖帶]]）。
4.  **給 100% 氧氣**：與 DCS 急救共用氧套組（見 [[減壓病徵兆識別與氧氣急救]]）；採**半坐臥**姿勢通常較平躺易呼吸 [4]。
5.  **立即送醫**：即使症狀在岸上很快緩解，**仍須就醫評估**——IPE 可復發，且需排除心臟因素 [1][3]。

> ❌ **禁忌**：不可「休息一下再下一支」；不可平躺硬撐；不可只當嗆水處理。

## 🛡️ 4. 風險因子與預防（側掛/技潛視角）

case-control 研究指出的獨立風險因子：**年齡 > 50 歲、女性、潛前服用 NSAID（非類固醇消炎止痛藥）、深度 > 20 m、潛前/潛中劇烈費力、每日常規用藥、（未控制之）高血壓** [3]。

側掛與技潛的可控預防：
1.  **配重與浮力正確**：過重配重 + BCD 操作不良 → 持續費力 → 抬高風險。做好 [[水中浮力與水平配平 (Trim)]] 與 [[防寒衣選擇與浮力配平影響]] 的配重檢查，減少無謂體能消耗 [1]。
2.  **避免潛前過度水合與 NSAID**：過度灌水與消炎止痛藥皆列為風險因子 [3]。
3.  **控制費力與逆流**：強流奮力游動使胸腔壓力驟升；技潛以 DPV/計畫化路線降低費力（見 [[DPV 水下推進器與側掛整合]]）。
4.  **保暖**：冷水是誘因，見 [[失溫預防與應變管理]]。
5.  **高風險族群體檢**：> 50 歲、高血壓、心臟病史者，轉技潛前應做潛水醫學評估 [1][3]。

> 📌 目前**無法事前預測個人易感性**（尚無篩檢工具）[1]——因此「識別 + 立即處置」比「預測」更關鍵。
