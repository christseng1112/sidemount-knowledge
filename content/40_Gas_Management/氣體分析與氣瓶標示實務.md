# 氣體分析與氣瓶標示實務 (Gas Analysis & Cylinder Labeling)

「**你沒親自分析的氣體，就不是你的氣體**」——呼吸錯誤氣體（高氧下深、缺氧混氣在淺處）是技術潛水最致命的錯誤類型之一，而防線只有兩道：**入手時分析**、**下水前與切換時讀標確認**。本篇整理氧氣分析儀操作 SOP 與氣瓶標示標準。

---

## 🔬 氧氣分析儀操作 SOP [1][2]

1.  **開機穩定**：開啟分析儀，等待讀值穩定（電化學感測器需暖機）。
2.  **校準 (Calibrate)**：依原廠指示以**空氣（20.9%）**為基準校準。**每個分析場次前都要校準**；儀器關機、移地、溫濕度大變後須重新校準 [2]。
3.  **接瓶測量**：分析儀接瓶閥或經流量限制器，**緩慢開閥、維持穩定流量**，等讀值穩定。
4.  **記錄**：記下實測 O₂%。
5.  **逐瓶分析**：**每一支瓶都要各自分析**——同一站充的氣仍會因殘氣、充填順序、頂壓差異而不同 [2]。
6.  **計算 MOD 並標示**：依實測值（非訂購值）計算 MOD，寫上 O₂%、MOD、日期、姓名縮寫 [2]。

> 📌 MOD 計算（公制）：$\text{MOD (m)} = \left(\frac{\text{ppO}_2\text{ 上限}}{\text{FO}_2} - 1\right) \times 10$。ppO₂ 上限工作氣 1.4 bar / 減壓氣 1.6 bar 之依據見 [[氧暴露與氮醉管理 (CNS、OTU、END)]]。

> 📋 **感測器壽命**：電化學 O₂ 感測器為消耗品（典型壽命 1–3 年），校準時若空氣讀值漂移大或反應遲緩即應更換。分析儀本身也要保養（見 [[側掛裝備日常保養與維護指引]]）。

## 🏷️ 氣瓶標示標準（WKPP/DIR 體系）[3]

| 標示 | 位置 | 規格 |
| :--- | :--- | :--- |
| **MOD 大字** | 瓶身兩側水平書寫 | 約 3 吋（7.6 cm）高的大數字，隊友一眼可讀 [3] |
| **MOD 瓶底** | 瓶底 | 前後縱隊時從後方可辨識 [3] |
| **分析標籤** | 瓶頸附近 | 實測 O₂%（混氣寫 O₂/He，如 "18/45"）、分析日期、姓名縮寫 [3] |
| **姓名** | 瓶身 | 團隊多瓶時識別歸屬 [3] |
| **純氧瓶特別規則** | 瓶身 | 標 **"20" 加寫 "OXYGEN"**——防止 "100" 被誤讀為 "70"/"120"；MOD 20 呎（公制體系標 **6 m**） [3] |
| **雜物清除** | 全瓶 | 撕除無關貼紙，避免誤讀 [3] |

> ⚠️ 台灣/公制環境注意：WKPP 標準以英呎標 MOD（如 EAN50 標 "70"）。公制團隊應統一以**米**標示（EAN50 → "21"、O₂ → "6"），**潛前簡報必須確認全隊單位一致**，混用單位比不標更危險。

---

## 🔁 標示與切換的閉環

標示的意義在氣體切換時兌現——切換程序（NO TOX 三確認：讀 MOD 標示 → 追管到一級頭 → 隊友交叉確認）詳見 [[側掛氣體切換標準程序]] 與 [[多氣瓶與減壓瓶掛載配置 (Technical Sidemount)]]。WKPP 流程要求：定位正確 MOD 瓶 → 部署二級頭 → 開閥看 SPG 壓降確認對應關係 → 隊友 OK 後才入嘴 [3]。
