# 氣體調配與高壓充填計算指引 (Gas Blending & Oxygen Safety)

在技術側掛潛水與深潛活動中，使用高氧（Nitrox）與三元混合氣（Trimix）是標準配置。氣體的精確調配與充填安全，直接關係到水下生命安全。

本篇將介紹高壓氣體調配的分壓法（Partial Pressure Blending）物理原理、氣體壓縮性偏差，並嵌入一個**「互動式高壓氣體調配計算機」**，最後說明極為關鍵的高壓氧氣安全與氧氣清洗（O₂ Clean）標準 [1][2]。

---

## ⚖️ 1. 分壓調配物理學：理想氣體與實氣偏差

分壓調配法是將純氧、純氦以及過濾空氣依次充入氣瓶中，利用分壓原理達到目標比例 [1]。

### 理想氣體分壓原理 (Dalton's Law)
根據道耳吞分壓定律，混合氣體的總壓力等於各組成分壓之和 [1]：
$$P_{total} = P_{O_2} + P_{He} + P_{N_2}$$
而在理想狀態下，充入的氣壓與氣體摩爾量成正比。

### 實氣偏差與壓縮係數 (Compressibility Factor Z)
在超過 200 bar 的高壓下，氣體分子間的作用力與分子自身體積會導致其偏離理想氣體狀態（即 $PV = nZRT$） [2][3]：
*   **氦氣 (Helium)**：氦分子極小，在高壓下較難壓縮（$Z > 1$）。如果按理想氣體計算充入 70 bar 的氦氣，當最終壓力達到 200 bar 時，實測的氦氣比例會**低於預期**（通常偏低 1–3%） [2][3]。
*   **氧氣與氮氣 (Oxygen & Nitrogen)**：高壓下相對較易壓縮（$Z < 1$）。
*   **調配實務**：高壓充填時，氣體會發熱膨脹。充填後必須等待氣瓶冷凍/冷卻至環境溫度（20°C）後，以實測壓力與氣體分析儀（He/O₂ 分析儀）數值為準進行校正 [2][4]。

function calculateBlending() {

  var fO2Start = parseFloat(document.getElementById('curO2').value) / 100.0;
  var fHeStart = parseFloat(document.getElementById('curHe').value) / 100.0;
  var pTarget = parseFloat(document.getElementById('tgtPress').value);
  var fO2Target = parseFloat(document.getElementById('tgtO2').value) / 100.0;
  var fHeTarget = parseFloat(document.getElementById('tgtHe').value) / 100.0;
  
  var resultBlock = document.getElementById('blendResultBlock');

  if (isNaN(pStart) || isNaN(fO2Start) || isNaN(fHeStart) || isNaN(pTarget) || isNaN(fO2Target) || isNaN(fHeTarget) ||
      pStart < 0 || fO2Start < 0 || fHeStart < 0 || pTarget <= pStart || fO2Target < 0 || fHeTarget < 0) {

    alert('氧氣與氦氣比例總和不能超過 100%！');
    return;
  }
  
  // Formulas:
  // 1. Added Helium pressure
  var pHeAdd = (pTarget * fHeTarget) - (pStart * fHeStart);

  var pAirAdd = ((pTarget * fN2Target) - (pStart * fN2Start)) / 0.79;
  
  // 3. Added Oxygen pressure
  var pO2Add = pTarget - pStart - pHeAdd - pAirAdd;
  
  resultBlock.style.display = 'block';
  warningDiv.style.color = '#ff9f43';
  warningDiv.innerText = '⚠️ 充氧或充氦順序可依潛店管路配置微調，高壓純氧充填速度務必極度緩慢！';
  
  if (pHeAdd < -0.01 || pAirAdd < -0.01 || pO2Add < -0.01) {
    document.getElementById('resCurPress').innerText = pStart + ' bar';
    document.getElementById('resAddHe').innerText = '無法計算';
    document.getElementById('resAddO2').innerText = '無法計算';
    document.getElementById('resAddAir').innerText = '無法計算';
    warningDiv.style.color = '#ff4a4a';
    warningDiv.innerText = '🚨 無法直接充填！目標混合氣中的氦、氮或氧分壓低於當前殘氣所提供的量。請先將氣瓶洩氣（Vent）至更低壓力再行計算。';
    return;
  }
  
  pHeAdd = Math.max(0, pHeAdd);
  pAirAdd = Math.max(0, pAirAdd);
  pO2Add = Math.max(0, pO2Add);

  var step2 = step1 + pHeAdd;
  var step3 = step2 + pO2Add;
  var step4 = pTarget; // top-up with air

## 🧯 3. 高壓氧安全與氧清洗 (Oxygen Safety & O₂ Cleaning)

調配 Nitrox 或 Trimix 時，經常需要將**純度高達 99% 的高壓氧氣**注入氣瓶中。高壓氧氣是強氧化劑，遇微量碳氫化合物（如普通潤滑油、油脂、皮膚碎屑）即可引發劇烈爆炸 [1][4]。

### 氧氣清洗標準 (Oxygen Clean / O₂ Clean)
任何接觸高濃度氧氣（分壓 $>40\%$ 或氧分壓 $>0.4$ bar）的硬體設備，必須進行嚴格的氧氣清洗 [4]：
1.  **徹底脫脂**：使用專用的去油溶劑（如高純度異丙醇或氧安全脫脂劑）清洗一級頭、氣瓶內部、瓶閥螺紋，完全清除工業油脂或手汗 [4][6]。
2.  **更換氧相容 O-rings**：將普通丁腈橡膠（Buna-N）O-rings 更換為氟橡膠（Viton）或聚氨酯材質 O-rings [4]。
3.  **使用氧相容潤滑劑**：**絕對禁止**使用普通矽油（Silicon grease）；必須使用氟素潤滑油（如 **Christo-Lube MC111** 或 Krytox） [4][6]。

### 高壓氧氣安全充填 SOP (Safeguards)
*   **慢開閥門**：開啟氧氣瓶或充排氣橋（Deco bridge）閥門時，務必**極度緩慢**。否則高速流動的氧氣在撞擊死角時會產生「絕熱壓縮（Adiabatic Compression）」，使局部溫度瞬間飆升至數百度，引燃金屬或密封圈 [1][4]。
*   **防塵防異物**：充排氣軟管接頭不使用時，必須蓋上防塵蓋，防止空氣中的灰塵或油污進入管路。
*   **充填速率限制**：分壓充填純氧時，流速限制在 **$5\text{ bar/min}$ (約 $70\text{ psi/min}$)** 以下，防止摩擦生熱 [1][4]。
