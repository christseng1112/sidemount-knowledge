# 側掛配件與口袋系統配置 (Sidemount Accessories & Pocket Configuration)

側掛潛水員的身體兩側被氣瓶完全佔據。這意味著傳統潛水衣上的「大腿外側口袋」在掛上氣瓶後會被完全遮擋，且大腿口袋的厚度會干涉氣瓶的平行貼合。因此，側掛發展出了獨特的**配件固定與口袋收納哲學：一切配件必須融入人體流線型，無懸吊物，且核心安全裝備必須雙手皆可觸及**。

本篇將詳細解析屁股口袋（Butt Pouch）的配置、切割工具的定位、主/備用手電筒的收納，以及浮力袋（DSMB）與線輪（Spool）的流線型固定。

---

## 📂 屁股口袋 (Butt Pouch / Utility Pouch) 的收納配置

屁股口袋是側掛潛水員的「移動工具箱」，主要收納潛水過程中可能需要、但**不具備即時緊急性**的備用裝備 [1][2]。

```
                 [ 臀部板 / 後腰 D 環 ]
                    │            │
             [雙頭扣]            [雙頭扣]
              ┌────────────────────────┐
              │     屁股口袋 (Pouch)    │  <--- 懸掛於臀部後下方
              │ ─ 備用面鏡 / 濕筆記      │
              │ ─ 備用 Bungee / 箭頭線標│
              └────────────────────────┘
```

*   **固定位置**：使用兩個不鏽鋼雙頭快拆扣（Double Enders），將口袋掛在**臀部板（Butt Plate）的D環、或是後腰帶的兩個D環上**。口袋應懸掛於臀部正後方、大腿與臀部交界處的凹陷區，這是水中阻力最小的盲區 [1][2][3]。
*   **收納物品明細**：
    *   備用面鏡（低容積摺疊式）。
    *   備用彈性繩（Spare Bungee）與備用快拆扣。
    *   水下記事本（Wet Notes）。
    *   洞穴/沉船用線標（Arrows / Non-directional Markers） [3][4]。
*   **物理要求**：口袋必須使用扁平拉鍊設計，不使用時呈扁平狀。嚴禁裝得過於臃腫，否則在穿越窄縫時會卡住，或是導致臀部產生不必要的下沉阻力 [1][5]。

## ✂️ 安全切割工具的定位 (Cutting Tools)

在封閉環境或有廢棄漁網的海域中，纏繞是極致命的威脅。切割工具的配置遵守**「雙手可及、多重備份」**的原則：

*   **位置要求（Ambidextrous Access）**：切割工具必須定位在**左手與右手都能輕易單手拔出**的位置。若一側肩膀被漁網纏繞無法動彈，另一手必須能立刻自救 [2][5]。
*   **推薦固定點**：
    1.  **腰帶前方（肚臍兩側）**：將線剪（Line Cutter）或割繩刀固定在腰帶前方。這是雙手最容易交會的自然區域 [2][5]。
    2.  **電腦錶/腕錶帶上**：將小型割繩器直接綁在手腕電腦錶的錶帶上，拔取極為直觀。
    3.  **前胸肩帶**：將鈦合金潛水剪刀（Shears）套夾在肩帶上，以橡皮圈防脫落 [2]。

## 🔦 主燈與備用手電筒 (Lights Configuration)

### 1. 分體式主燈 (Canister Primary Light)
*   **電池罐 (Canister)**：通常掛在臀部右側下方的腰帶上（緊鄰腰帶扣），或利用不鏽鋼夾平貼固定於背後。
*   **燈頭 (Light Head)**：使用 Goodman 手把固定於**左手背**上。這允許右手空出來操作充排氣閥、掛卸氣瓶或進行隊友共氣 [3][4]。

### 2. 備用手電筒 (Backup Lights) - 經典側掛收納法
側掛有一套非常標準且流線的備用手電筒固定方式：
*   **固定方法**：在手電筒尾端綁上一個 Bolt snap。利用兩個軟橡膠圈（Snoopy Loops 或 O-ring）將手電筒**平行平貼固定在肩膀前側的肩帶上** [3][5]。
*   **走線規範**：燈頭必須**朝下**指向潛水員的腰部，並用 Bolt snap 扣在肩帶 D 環上。
*   **物理優勢**：
    *   燈頭朝下能防止意外開機時光線直射隊友眼睛造成眩光。
    *   手電筒平貼在胸前，完全沒有飄動空間，不增加阻力。
    *   一旦主燈故障，潛水員只需用單手向下一拉，即可輕鬆解開安全扣並開啟備用手電筒 [3][5][6]。

## 🎈 浮力袋與線輪的收納 (DSMB & Spool)

*   **浮力袋 (DSMB)**：折疊平整並用彈性繩紮緊。通常固定在**氣囊（Wing）的最下端中線上**、或是利用快拆扣平貼掛在背部下方。嚴禁將浮力袋掛在身側隨水流飄動 [3][5]。
*   **線輪 (Spool)**：使用雙頭快拆扣，扣在臀部左側的 D 環上。線輪必須緊貼身體，不可像鐘擺一樣 dangling 懸吊 [3][5]。

## 🎯 Razor 4.0 系統的配件系統特點

當您使用 **[[Razor 側掛系統調整與配置|Razor 4.0]]** 系統時，配件固定有專屬的設計配件：

*   **Razor Utility Pocket (專用屁股口袋)**：這是一款極扁平且帶有內部彈性繩分類設計的口袋。它透過特製的三角形快拆扣平貼在下背部，在穿越窄縫時能提供極佳的防撞防脫落保護 [2][10]。
*   **胸前備用手電筒固定**：Razor 系統隨附專用的 Snoopy Loops，能將備用手電筒以極高的緊度鎖定在肩帶上，完美展現極簡主義 [10]。
