# 主流側掛系統橫向對比 (Hollis Katana 2、Dive Rite Nomad、Apeks WSX)

本庫已有 [[XDEEP Stealth 2.0 系統調整與配置]] 與 [[Razor 側掛系統調整與配置]] 兩篇深度專文。本篇補上市場上其他三個主流選項，並給出「先選流派、再選品牌」的決策框架 [1]。

---

## 🗺️ 先懂兩大流派

| | **墨西哥式 (Mexican Style)** | **佛州式 (Florida Style)** |
| :--- | :--- | :--- |
| **原生環境** | 溫暖淡水洞穴、鋁瓶 | 冷水高流量泉洞、鋼瓶 |
| **氣囊形狀** | 小升力、貼背、低調 | 大升力、沿脊椎分佈 |
| **背帶** | 可變長度、分離式背板 | 固定背板、類 Hogarthian |
| **代表** | Razor、Apeks WSX | Dive Rite Nomad、Halcyon Contour |
| **跨界混血** | XDEEP Stealth、Hollis Katana 2（冷水設計者做的墨式外型，鋼鋁通吃）[1] | |

**選擇順序：你的氣瓶（鋁/鋼）與防寒衣（濕/乾）→ 流派 → 品牌細節** [1]。氣瓶浮力特性見 [[鋼瓶與鋁瓶的浮力與配重管理]]。

---

## ⚔️ 三系統重點

### Hollis Katana 2
*   **定位**：XDEEP Stealth Tec 的直接競品——規格、價位、功能高度重疊，差異在細節 [2]。
*   **強項**：背帶可在 **Y 型與 H 型之間改裝**，肩帶調整幅度大、上手快，**教學與租賃友善** [3]；氣囊剪裁鋼鋁瓶皆可壓平。
*   **注意**：配重整合槽位於背板脊椎袋，大量配重時重心偏高，需與瓶位微調搭配（原理同 [[水中浮力與水平配平 (Trim)]]）。

### Dive Rite Nomad（XT / LTZ / Ray 世代）
*   **定位**：佛州式經典。**XT** 為冷水鋼瓶大升力工作馬；**LTZ / Ray** 是向墨式市場靠攏的輕量化後繼 [1][3]。
*   **強項**：耐用度口碑極佳；Ray 世代有**免工具彈性繩長度調整**與氣囊內拉鍊配重袋 [3]；Transpac 背帶系統可與單瓶/雙瓶背掛共用，肌肉記憶一致 [3]。
*   **注意**：XT 對嬌小體型與純鋁瓶熱帶潛水偏笨重——那是 LTZ/Ray 的場景。

### Apeks WSX（WSX-25 / WSX-45）
*   **定位**：墨式極簡路線的大廠量產版，WSX-25 主打單瓶/雙鋁瓶休閒側掛，WSX-45 上探技術用途 [1]。
*   **強項**：模組化織帶＋護布翻蓋（fabric-flap）包覆氣囊，流線且抗磨；大廠通路保固完整，全球服務點多。
*   **注意**：極簡＝可調整幅度較 Katana 2 小；重型鋼瓶多瓶配置非其設計目標。

## 🧭 決策建議

1.  **台灣東北角/冷水鋼瓶為主** → Nomad XT 或 Katana 2（乾衣情境見 [[防寒衣選擇與浮力配平影響]]）。
2.  **熱帶旅行鋁瓶為主** → WSX-25、Nomad Ray，或本庫已詳述的 Razor / Stealth。
3.  **教學/多人共用/還不確定** → Katana 2 的調整幅度最寬容。
4.  無論選哪套，**購入後的瓶位與彈性繩調校**才是決定水中表現的主因——調校方法論同 [[XDEEP Stealth 2.0 系統調整與配置]] 篇的流程。二手購入檢查見 [[二手側掛裝備選購指南]]。
