# 氣瓶懸掛系統與硬體配置 (Sidemount Cylinder Rigging & Hardware)

氣瓶懸掛系統（Cylinder Rigging）是側掛潛水最核心的硬體技術。其配置的精準度，直接決定了氣瓶在水中是否能與潛水員身體平行、游動時是否晃動（鐘擺效應），以及在水下進行掛卸氣瓶時的流暢度。

本篇將詳細介紹 Rigging 的核心組件、**瓶底快拆扣的黃金高度計算與 45 度扭力定位法**、瓶首 Bungee 的運作機制，以及鋼瓶與鋁瓶的 Rigging 實務差異。

---

## ⚙️ Rigging 系統核心組件

一組標準的側掛氣瓶懸掛套件（Rigging Kit）包含以下硬體：

```
【氣瓶頂部 (Neck)】                      【氣瓶底部 (Butt)】
    [瓶閥/閥手輪]                            [氣瓶夾 Cam Band / Metal Clamp]
        │                                         │
    [瓶首彈性繩 Bungee]                      [尼龍繩繩圈 Rigging Leash]
                                                  │
                                             [瓶底快拆扣 Bolt Snap]
```

1.  **氣瓶夾 (Cylinder Clamp / Cam Band)**：
    *   **鋁瓶**：多使用不鏽鋼金屬喉箍（Worm Gear Clamp），外側套上耐磨橡膠管保護套，以防割傷防寒衣或刮傷手 [1]。
    *   **鋼瓶**：多使用標準的織帶快拆夾（Cam Band），調整位置較為快速。
2.  **瓶底快拆扣 (Butt Bolt Snap)**：通常使用 **3/8 吋大眼睛不鏽鋼單頭快拆扣**。大眼睛的設計是為了在戴著厚手套、或是極度寒冷手指僵硬時，依然能以單指流暢操作掛卸 [1][2]。
3.  **尼龍繩繩圈 (Rigging Leash/Cord)**：用於將 Bolt snap 綁定在氣瓶夾上。不可使用金屬快拆環或金屬鍊，必須使用可切斷的尼龍繩，以防在水下發生極端勾掛時，能以割繩刀切斷氣瓶自救 [2][3]。

## 📐 瓶底快拆扣高度計算與 45 度扭力定位法

瓶底快拆扣的位置是控制氣瓶水中角度的關鍵。必須經過精確的幾何測量與綁定：

### 1. 黃金比例高度計算 (Rigging Geometry)
*   **氣瓶夾位置**：以最常見的 **AL80 鋁瓶** 為例，不鏽鋼氣瓶夾的安裝位置應距離氣瓶底部向上約 **3 至 4 指寬 (約 8-10 cm)** [4][5]。
*   **快拆扣懸掛高度**：利用尼龍繩將 Bolt snap 綁緊在氣瓶夾上。**綁好後，Bolt snap 的尾端（滑動開關處）應剛好與氣瓶平底切齊，或是比底面高出 1 cm** [4][5][6]。
    *   **繩太長（鐘擺效應）**：如果繩圈過長，瓶底會像鐘擺一樣左右晃動，游動時會不斷撞擊大腿，且氣瓶無法保持流線型 [6]。
    *   **繩太短（掛扣困難）**：如果繩圈過短，水下操作時空間不足，會極難將快拆扣掛上腰部 D 環 [6]。

```
【正確 Rigging 高度】                  【錯誤 Rigging (過長)】
      │                                    │
    [氣瓶夾]                             [氣瓶夾]
      │                                    │
    [繩圈 (緊湊)]                        [繩圈 (過長)] ──> 導致氣瓶水中飄移晃動
      │                                    │
   [快拆扣]                                │
   ┌──┴──┐                               [快拆扣]
   │氣瓶底│                                 │
   └─────┘                              ┌──┴──┐
                                        │氣瓶底│
                                        └─────┘
```

### 2. 45 度扭力定位法 (Torque Control)
將瓶底快拆扣綁在氣瓶夾上時，其相對於瓶閥的方向有嚴格的要求：
*   **定位角度**：將瓶閥的氣體出氣口朝向正前方，此時瓶底的尼龍繩固定點應位於**與出氣口呈 45 度夾角的方向**上 [6][7]。
*   **物理機制**：當您在水下將瓶底快拆扣掛在腰帶/臀部 D 環上時，這個 45 度夾角會對氣瓶產生一個向內旋轉的力矩（Torque）。這個力矩會迫使氣瓶自動往潛水員的身體中線平貼，防止氣瓶向外側飄移翻轉（Flaring），維持完美的流線型 [6][7]。

## 🔄 瓶首懸掛機制：Bungee 系統比較

瓶首（閥門處）的懸掛主要依賴背帶上的彈性繩（Bungee）。目前主流有三種流派：

1.  **圈狀彈性繩 (Loop Bungee)**：
    *   *機制*：如 **[[Razor 側掛系統調整與配置|Razor 4.0]]** 採用的系統。彈性繩呈圈狀固定在肩帶上，水下直接繞過整個瓶閥手輪將其扣緊。
    *   *優點*：瓶閥縮入腋下的效果最好，流線型極佳，且可以利用 Bungee Tri-glide 隨時調整張力 [8][9]。
2.  **連續性彈性繩 (Continuous Bungee)**：
    *   *機制*：一條長彈性繩橫跨背部，兩端帶有快拆扣。下水前將快拆扣扣在瓶首的繩圈（Choker）上。
    *   *優點*：左右拉力對稱，當單側氣瓶卸下時，彈性繩不會縮回背後 [9]。
3.  **瓶頸收緊帶 (Choker / Neck Clip)**：
    *   *機制*：在氣瓶頸部用織帶套緊，並連接著一個 Bolt snap。將此 Bolt snap 直接扣在肩帶的 D 環上，再用彈性繩做輔助固定。
    *   *優點*：提供絕對堅固的物理金屬連接，適合在浪大、流急的海域進行船潛，防止氣瓶在水面跳動時鬆脫 [3][9]。

## ⚖️ 鋁瓶與鋼瓶的 Rigging 差異

氣瓶材質不同，其水中重量與浮力特性完全不同，因此 Rigging 配置必須做出修正：

*   **鋁瓶 (AL80) Rigging 特點**：
    *   *氣瓶夾材質*：必須使用不鏽鋼管夾（Worm Gear Clamp），因為鋁瓶瓶身較滑，織帶夾容易滑動。
    *   *位置*：氣瓶夾定位較高（距離底端約 8-10 cm），以補償鋁瓶變輕後瓶底的上浮 [1][10]。
    *   *滑動 D 環需求*：必須搭配**滑動 D 環（Sliding D-rings）**，以便在水下將上浮的瓶底往前掛扣 [5][10]。
*   **鋼瓶 Rigging 特點**：
    *   *氣瓶夾材質*：使用織帶夾（Cam Band）即可。
    *   *位置*：鋼瓶全程為負浮力（尤其是瓶底很重），為了防止「腳下沉」，**鋼瓶的氣瓶夾位置應下移（距離底端約 3-5 cm）**。這將掛點往瓶底推，相當於將氣瓶往上提，能有效平衡鋼瓶的下沉力 [5][10]。
    *   *D 環需求*：鋼瓶不需要滑動 D 環，全程掛在臀部後方的固定 D 環即可 [5]。
