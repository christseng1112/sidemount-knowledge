# 側掛動力推進技巧 (Sidemount Propulsion Techniques)

在側掛潛水中，傳統的自由式踢法（Flutter Kick）是被嚴格禁止的。因為自由式踢法會產生向下的水流，在洞穴或沉船中會瞬間將底部的泥沙攪起（稱為 Silt-out / 揚沙），造成能見度歸零的致命險境 [1][2]。

側掛潛水員必須掌握一整套**水平面推進技巧**，以在不破壞水平 Trim、不攪動底沙、且不使用雙手的前提下，實現精確的前進、後退與原地轉向。

---

## 🐸 1. 蛙踢 (Frog Kick) — 主力推進技巧

蛙踢是技術潛水與側掛中最基礎、最高效的踢法。其物理特點是**將水流完全向後方推動，且包含一個高效的「滑行期」** [4][7]。

```
【蛙踢動作分解圖】
 1. 準備期 (Ready) ───> 2. 張開期 (Spread) ───> 3. 推水期 (Thrust) ───> 4. 滑行期 (Glide)
   雙腿併攏               雙膝外張 (約肩寬)         腳踝旋轉，腳掌      雙腿完全併攏伸直
   大腿平行               小腿微收，腳踝勾緊         向後弧形夾水        利用慣性無阻力滑行
```

*   **動作要領**：
    1.  **準備**：身體維持水平 Trim，雙腿併攏，大腿與脊椎共線。
    2.  **張開**：收縮大腿後側肌群，使小腿向上收起（膝蓋彎曲 90度），雙膝向外微微張開至約肩膀寬度。腳踝向上勾緊，蛙鞋面平行於底部 [7][8]。
    3.  **推水**：腳踝向外旋轉，使蛙鞋寬面朝向後方，大腿與小腿發力，雙腳掌向後下方做弧形「夾水」動作 [7][10]。
    4.  **滑行**：推水完成後，**雙腳必須立刻併攏、繃直，腳尖伸直**，將雙腿的阻力降到最低，利用推進的慣性向前無阻力滑行 2 - 3 秒。這是蛙踢最省氣的關鍵 [7][11]。

## 🔄 2. 改良式蛙踢 (Modified Frog Kick) — 限制空間推進

當進入狹窄的洞穴、沉船走廊或團體潛水排成縱隊時，標準蛙踢的外張動作會撞擊兩側壁面。此時必須使用改良式蛙踢 [13][14]。

*   **動作要領**：
    *   維持大腿完全不動（與脊椎鎖定呈水平線），雙膝距離拉近。
    *   **完全依靠膝蓋以下的關節彎曲，配合腳踝的微小旋轉**進行小幅度的「撥水」 [7][13]。
*   **物理效果**：迎水截面積降至最低，推進力溫和且高度可控，能以極小的動靜在窄道中前進，不碰觸任何周邊岩壁 [14][23]。

## 🔙 3. 後退踢法 (Backwards Kick / Back Kick) — 距離控制與撤退

後退踢法是側掛中難度最高、但極為關鍵的自救與定位技巧。它能讓潛水員在不轉身的情況下，平穩地向後倒退 [15][17]。

*   **動作要領**：
    1.  **初始狀態**：雙腿併攏伸直，腳踝放鬆。
    2.  **撈水準備**：腳踝向上勾起，使腳掌與大腿呈 90 度，蛙鞋寬面朝向前方。
    3.  **撈水推進**：大腿微彎，小腿向外/下方微微張開，利用腳踝與小腿肌肉發力，將蛙鞋面**向前、向內「撈水/摳水」**，將水流推向前方 [17][18]。
    4.  **回收**：推水後，雙腿放鬆伸直併攏，重複上述動作 [18][19]。
*   **物理效果**：產生向前的推力，使潛水員向後倒退。此技巧能讓您在拍照、觀察或面臨通道終點時，優雅地與障礙物保持安全距離，完全不需動手 [15][17]。

## 🛸 4. 直升機轉彎 (Helicopter Turn) — 原地 360° 轉向

直升機轉彎允許潛水員在不產生任何前後位移的情況下，以身體重心為軸心原地進行 360 度旋轉 [11][20]。

*   **動作要領**（以**順時針轉向**為例）：
    *   **左腿**：執行蛙踢的推水前半段動作（腳掌向外旋轉，將水向後推）。
    *   **右腿**：執行後退踢的撈水動作（將水向前推） [11][20]。
    *   兩腿動作協調配合，一推一拉，身體便會以腹部重心為圓心，原地順時針旋轉。
*   **物理效果**：在極窄的死胡同、或是手持相機/手電筒時，無需划手即可在原地輕鬆轉身撤退 [21][22]。

## ⚠️ 推進與 Trim 的協同張力控制

在踢動過程中，初學者常犯的錯誤是：踢腿時上半身隨著晃動（蛇行或上下擺動）。
*   **物理機制**：根據牛頓第三運動定律，踢腿產生的反作用力如果不被身體核心鎖定，就會轉化為軀幹的晃動。
*   **控制訣竅**：在踢動時，**從腰部到肩膀的肌肉必須保持鐵板般的緊繃**，將反作用力完全轉化為向前的線性動能。只有膝蓋、小腿與腳踝是靈活運動的，軀幹必須保持絕對的靜態水平 [9][20][23]。
