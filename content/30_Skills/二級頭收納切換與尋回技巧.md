# 二級頭收納、切換與尋回技巧 (Regulator Stowage, Switching & Recovery)

側掛的二級頭管理與背掛單瓶完全不同：**兩顆二級頭都是「主用」**，整趟潛水輪流呼吸；不呼吸的那顆必須收納在固定、可盲摸取回的位置。本篇整理標準收納結構、切換動作與「掉嘴」時的尋回程序。

---

## 🔧 標準收納結構（長管/短管配置）[1]

側掛最主流的配置為 **Long Hose / Short Hose**（休閒、技術、洞穴課程通用教法）：

| | 右瓶（長管） | 左瓶（短管） |
| :--- | :--- | :--- |
| **管長** | 2.1 m（7 ft） | 56–61 cm（22–24 吋） |
| **路由** | Hogarthian 式：沿胸前向上、逆時針繞頸後、由右側入嘴 [1] | 短管直上，掛於下巴下方 |
| **收納** | 不用時對折以 Bolt snap 扣右肩 D 環；餘管以橡皮圈收於左瓶身 [1] | **Bungee 項圈（Necklace）** 恆掛喉下 |
| **緊急角色** | 捐給 OOG 隊友（見 [[氣源耗盡 (OOG) 與長管分享程序]]） | 自己的最後備援 |

> 📌 **Break-away 收納**：長管扣點可用 O-ring 或細 Bungee 圈套在二級頭排氣罩上做「可扯脫」連接——緊急時直接拉扯即脫離，不需操作快扣 [2]。代價是可能意外脫落，屬個人風險取捨。

---

## 🔄 切換動作標準流程

側掛每 20–30 bar 輪換呼吸一次（理由與時機見 [[側掛氣體切換標準程序]]），因此切換動作一天會做數十次，必須練到反射化 [1]：

1.  **取**：單手取出目標二級頭（項圈或解扣長管）。
2.  **換**：吐出口中二級頭 → 放入新二級頭 → **按排水鈕或用力吐氣排水** → 恢復呼吸。
3.  **收**：原二級頭立即歸位（長管扣回右肩 D 環 / 短管本來就在項圈上）。**絕不允許二級頭懸盪**——懸盪的二級頭在側掛會垂進揚沙區、勾線、進泥。
4.  **確認**：呼吸 2–3 口確認供氣順暢，瞄一眼對應 SPG。

> ⚠️ 頻繁切換的隱藏安全效益：兩顆二級頭都持續被使用驗證，**確保任一顆在需要捐氣時是「已知可用」狀態**，也防止未使用側在高泥沙環境中卡死 [1]。

## 🔍 二級頭尋回（掉嘴應變）

側掛「掉嘴」的處境比背掛安全得多，因為**項圈備用永遠在喉下固定位置**：

### 標準反應（優先序）
1.  **直接取項圈二級頭**：低頭、下巴夾住 Bungee 位置，單手送入口中，排水呼吸。**這是側掛的第一反應**，不做背掛式的大範圍手臂掃管（Arm Sweep）。
2.  恢復呼吸後，再從容尋回脫落的長管：沿右瓶一級頭 → 順管摸至二級頭。
3.  若脫落的是項圈二級頭（Bungee 斷裂），改咬長管，項圈頭順左瓶管線尋回。

### 為什麼不先做 Arm Sweep？
背掛的掃管動作是因為備用位置不固定；側掛兩顆頭的「家」都是固定的（右肩 D 環、喉下項圈），**直接去固定位置拿永遠比盲掃快** [1][3]。

### 項圈製作要點 [3]
*   用約 60 cm（2 ft）的 **3 mm Bungee 繩**，長度以二級頭懸在**下巴正下方一拳內**為準——過長會盪到胸前、過短妨礙轉頭 [3]。
*   兩端各打**雙漁人結（Double Fisherman's Bend）**，兩結間留約 5 cm（2 吋）容納咬嘴——此結構天然具 Break-away 特性，需要時可直接扯脫 [3]。
