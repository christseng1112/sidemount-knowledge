# 側掛裝備旅行打包與航空託運 (Traveling with Sidemount Gear)

側掛是**旅行最友善的技術構型**：沒有背板重鐵、沒有雙瓶 manifold，一套 Razor/Stealth 級背帶加兩套調節器，比背掛雙瓶配置輕數公斤。但雙調節器、多 bolt snap、線軸剪刀這些東西，在打包與安檢上各有眉角 [1][2]。

---

## 🎒 手提 vs 託運的鐵律

**判斷式：遺失後這趟旅程就毀了的東西，上手提** [1]。

| 手提（隨身） | 託運 |
| :--- | :--- |
| **潛水電腦**（電池取出或關機防誤觸）[2] | 背帶＋氣囊（當緩衝材） |
| **調節器**（兩套，padded reg bag） | 蛙鞋（貼箱壁當防撞樑）[2] |
| 面鏡（硬盒）＋度數備用面鏡 | 防寒衣（蓋頂當緩衝）[2] |
| 證照卡、保險卡、藥品、日誌 | **刀、剪刀、線刀——一律託運**（手提必沒收）[1] |
| | 燈具（見下方電池規定） |

*   調節器託運亦可（原廠 padded bag 塞進 BCD 中央）[2]，但側掛**兩套調節器＝雙倍損失風險**，優先手提。
*   手提重量限額 7–10 kg 很容易爆：調節器兩套約 4 kg——電腦、面鏡分散到同行者或外套口袋。

---

## 📦 託運打包順序（防撞三明治）[2]

```
底層：背帶+氣囊攤平 ──> 兩側：蛙鞋直立貼壁 ──> 中央：重物/工具（毛巾包）
──> 上層：防寒衣攤平覆蓋 ──> 縫隙：襪子包小物塞滿
```

側掛特有項目：
*   **Bolt snap 與彈性繩**：全部留在背帶上沒問題，但**用織帶把鬆散的 snap 綁攏**——散裝金屬件在 X 光下常引來開箱檢查。
*   **線軸/捲線器**：線捲緊、用橡皮筋鎖死，避免箱內炸開纏繞所有東西。
*   **配重—絕不託運**：配重去當地借/租（幾乎所有店家免費或低價提供）。

---

## 🔋 電池與危險品規定

1.  **鋰電池（燈具、電腦、相機）**：備用鋰電池與行動電源**只能手提**，端子貼絕緣膠帶；裝在設備內的電池可託運但建議取出手提。大容量罐燈電池（>100 Wh）需航空公司核准——買燈時就查 Wh 數。
2.  **氣瓶不隨行**：目的地租瓶是常態。若攜帶私人小瓶（argon/suit bottle），必須**卸閥、開口、可目視內部**才收——多數情況不值得。
3.  分析儀的氧感測器（電化學）可正常手提/託運。

---

## ✈️ 額度與航空公司

*   經濟艙常規：託運 23 kg＋手提 7–10 kg [1]。一套完整側掛旅行裝備（含 5mm 濕衣）約 18–22 kg——**買個行李秤，打包完秤、回程再秤**（濕裝備會變重）[2]。
*   部分航空有**潛水裝備優惠額度**（如土耳其航空特定航線免費一套潛水裝備）[1]——訂票前查「sporting equipment / diving equipment」條款，通常需事先登記。
*   台灣出發常見航線（飛沖繩/菲律賓/馬來西亞）廉航額度買到 25–30 kg 通常比正價航空加購便宜——含來回濕裝備回程餘裕。

---

## 🧰 目的地檢查清單

1.  **落地第一件事**：開箱檢查兩套調節器外觀、旋鈕、錶面——運輸損傷當場拍照才有得談。
2.  組裝後在泳池/第一支 check dive 完整測試：正壓漏氣測試流程見 [[潛水前檢查清單 (Pre-dive Checklists)]]；行前保養時程見 [[側掛裝備日常保養與維護指引]]。
3.  租瓶閥規（DIN/Yoke、左閥可得性）提前信件確認——台灣與東南亞實況見 [[台灣側掛潛點適用性指南]]。
4.  帶最小維修包：O-ring 組、bungee 備線、束線帶、六角扳手——離島沒有潛水五金行。
