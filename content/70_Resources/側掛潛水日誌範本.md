# 側掛潛水日誌範本 (Sidemount Dive Log Template)

通用日誌記不到側掛最有價值的數據：**雙瓶各自的壓力消耗、輪換節奏、裝備調整參數**。本範本針對側掛設計——複製下方區塊到每篇日誌筆記即可。日誌的核心用途：累積 **RMV 資料庫**（見 [[呼吸率 (SAC & RMV) 計算指引]]）與**裝備參數收斂**。

---

## 📋 範本（複製使用）

```markdown
# Dive #___ ｜ 日期：2026-__-__ ｜ 潛點：______

## 基本剖面
- 最大深度：___ m ｜ 平均深度：___ m
- 潛水時間：___ min ｜ 水溫：___ °C ｜ 能見度：___ m
- 海況/流：______ ｜ 入水方式：岸潛 / 船潛（跨步 / 水中掛瓶）

## 氣體（側掛核心區）
| | 左瓶 | 右瓶 |
|---|---|---|
| 瓶種/容積 | AL80 11.1L | AL80 11.1L |
| 氣體（實測O₂%） | | |
| 起始壓力 (bar) | | |
| 結束壓力 (bar) | | |
| 消耗 (bar) | | |
- 輪換節奏：每 ___ bar 切換｜結束壓差：___ bar
- 本潛 RMV = (左Δ+右Δ)×水容積 ÷ (時間×平均ATA) = ___ L/min
- Stage/減壓瓶：種類___ 起/迄壓力 ___/___

## 裝備配置快照
- 背帶：______｜Bungee 張力/長度備註：______
- 配重：___ kg（位置：背脊/側袋/前袋）
- 防寒：______（內襯：______）
- 蛙鞋：______｜其他變動：______

## 技巧與演練

- 演練項目：氣閥演練 / 氣體切換 / SMB 部署 / 其他：______
- 卡點與改進：______

## Trim / 浮力自評（1-5）
- 水平姿態：___ ｜ 懸停穩定：___ ｜ 瓶位貼合：___
```

---

## 🧭 使用要領

1.  **「下次要改的一件事」是整本日誌最重要的欄位**——每潛只改一個變數（Bungee 高度、配重位置、瓶帶高度），才能歸因。裝備參數收斂後，快照區就只剩「同上」。
2.  **RMV 累積 10 支以上再取均值**使用於氣體計劃；平靜潛與高負荷潛（逆流、任務多）分開統計（見 [[呼吸率 (SAC & RMV) 計算指引]] 的任務載荷表）。
3.  **結束壓差**持續 >30 bar 代表輪換紀律有問題，回讀 [[側掛氣體切換標準程序]]。
4.  電腦錶匯出（Subsurface 等，見 [[減壓規劃軟體與梯度因子]]）記剖面，本範本記**電腦記不到的東西**：裝備參數、技巧自評、改進項。
5.  Obsidian 用法：每潛一篇筆記，tag `#divelog`，可用 Dataview 統計 RMV 趨勢。
